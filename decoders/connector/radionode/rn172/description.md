# RN172WCD by RADIONODE®: Versatile Wi-Fi Sensor Data Transmitter

The RN172WCD by RADIONODE® is a versatile **Wi-Fi sensor data transmitter** designed for **real-time environmental monitoring** in industrial, commercial, and laboratory settings. It supports a wide range of UA series sensors, including gas detectors (Co2, NH3, etc.), thermal sensors (PT100, thermocouples), and analog transmitters (4–20 mA, 0–1 V), enabling flexible deployment across various applications. With Wi-Fi (IEEE 802.11 b/g), MODBUS TCP, and HTTP/HTTPS connectivity, the device seamlessly transmits data to cloud platforms like Radionode365, local servers, or PLCs for centralized monitoring. Additional features include a built-in buzzer, dual-color LED indicators, and a 4-digit display for real-time readings and alerts. The RN172WCD also supports remote configuration via Telnet and offers robust alarm functionalities, including SMS and voice call notifications, making it an ideal solution for safety-critical environments such as gas monitoring, HVAC systems, and industrial automation.
